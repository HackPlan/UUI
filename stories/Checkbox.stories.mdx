# 多选框 Checkbox

在一组选择项中进行多选。

### 简单使用

<Preview>
  <Story name="Checkbox">
    {() => {
      const [flag, setFlag] = useState(false)
      return (
        <Checkbox
          label={'Apple'}
          value={flag}
          onChange={(value) => { console.log(value); setFlag(value) }}
        />
      )
    }}
  </Story>
</Preview>

### 不可用状态

<Preview>
  <Story name="Disabled Checkbox">
    {() => {
      const [flag, setFlag] = useState(false)
      return (
        <div>
          <Checkbox
            className="mr-2"
            label={'Apple'}
            value={flag}
            onChange={(value) => { console.log(value); setFlag(value) }}
            disabled
          />
          <Checkbox
            className="mr-2"
            label={'Banana'}
            value={!flag}
            onChange={(value) => { console.log(value); setFlag(value) }}
            disabled
          />
        </div>
      )
    }}
  </Story>
</Preview>
