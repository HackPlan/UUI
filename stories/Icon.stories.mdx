# 图标 Icon

渲染图标工具，这个组件本身没有任何预置的图表库。

> 文档演示图片来自网络： https://www.iconfont.cn/collections/detail?cid=15329


### 多种数据源导入为图标

```typescript
const icon1 = require('./assets/icon_xxx.png');
import icon2, { ReactComponent as icon3 } from './assets/icon_xxx.svg';
/**
 * icon1: string
 * icon2: string
 * icon3: SvgrComponent
 */
```
