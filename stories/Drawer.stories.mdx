# 抽屉 Drawer

从屏幕边缘滑出的浮层面板。

### 简单使用

<Canvas>
  <Story name="Drawer"> {
    ()=>{
      const [active, setActive] = useState(false)
      return (
        <div>
          <Button onClick={() => { setActive((value=> !value)) }}>Click me!</Button>
          <Drawer
            active={active}
            onDismiss={() => {
            setActive(false)
            }}
          >
            Content
          </Drawer>
        </div>
      )
    }
  } </Story>
</Canvas>
