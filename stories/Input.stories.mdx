# 输入框 Input


## 文本输入框 TextField

### 简单使用

<Preview>
  <Story name="TextField">
    {() => {
      const [text, setText] = useState(null)
      return (
        <TextField
          type='text'
          value={text}
          onChange={(value) => { setText(value) }}
          placeholder={'Input text...'}
        />
      )
    }}
  </Story>
</Preview>

### 限制文本输入长度

<Preview>
  <Story name="TextField with max length limit">
    {() => {
      const [text, setText] = useState(null)
      return (
        <TextField
          type='text'
          value={text}
          maxLength={10}
          showLengthIndicator
          onChange={(value) => { setText(value) }}
          placeholder={'Input text...'}
        />
      )
    }}
  </Story>
</Preview>
