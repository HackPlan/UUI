# UUI Customize Tutorial

UUI 从设计开发之初，就考虑到要给使用者提供高度的组件样式自定义功能。

在实际使用 UUI 的组件时，我们有多种方法去重写组件的样式。

## 外部 CSS 文件

UUI 的组件每一个 Node 都配有一个 className，比如 `UUI-Stepper-Root`、`UUI-Stepper-MinusButton` 等等。
有了这些 className，我们就可以用类选择器来修改组件的样式。

例如，我们创建一个 `UUIStepperCustomize.scss` 的样式文件用来修改 Stepper 的默认样式，并导入到页面文件：

```scss
.UUI-Stepper-Root {
  .UUI-Stepper-MinusButton, .UUI-Stepper-PlusButton {
    background-color: #f5f7fa;
    color: #606266;
  }
  .UUI-Stepper-MinusButton, .UUI-Stepper-PlusButton, .UUI-Stepper-Input {
    border: 1px solid #dcdfe6;
  }
  &:not(.ControlRight) {
    .UUI-Stepper-MinusButton {
      border-top-left-radius: 4px;
      border-bottom-left-radius: 4px;
    }
    .UUI-Stepper-PlusButton {
      border-top-right-radius: 4px;
      border-bottom-right-radius: 4px;
    }
  }
  &.ControlRight {
    .UUI-Stepper-MinusButton {
      border-bottom-right-radius: 4px;
    }
    .UUI-Stepper-PlusButton {
      border-top-right-radius: 4px;
    }
    .UUI-Stepper-Input {
      border-top-left-radius: 4px;
      border-bottom-left-radius: 4px;
      overflow: hidden;
    }
  }
}
```

## 添加新的 className

可以配合 tailwindcss 来修改组件样式

```jsx
<Slider
  value={value}
  max={100}
  min={0}
  step={1}
  className={''}
  onChange={(value) => setValue(value)}
  customize={{
    ActiveLine: {
      extendClassName: 'bg-blue-500'
    },
    Thumb: {
      extendClassName: 'border-blue-500 rounded-none'
    },
  }}
/>
```

## 内联样式 style

```jsx
<Switch
  value={checked}
  onChange={(value) => { setChecked(value); }}
  customize={{
    Button: {
      Root: {
        extendStyle: {
          backgroundColor: checked ? '#4299e1' : 'rgba(0,0,0,.25)',
          border: 'none',
        }
      }
    },
    Thumb: {
      extendStyle: {
        backgroundColor: 'white',
      }
    }
  }}
>
</Switch>
```

## 添加额外的 Elements

```jsx
<TextField
  type='text'
  value={text}
  onChange={(value) => { setText(value) }}
  placeholder={'Input money...'}
  customize={{
    Root: {
      extendStyle: {
        padding: '0 0.5rem',
        border: '1px solid #dcdfe6',
        borderRadius: 4,
      },
      extendChildrenBefore: (
        <div className="text-gray-600">$</div>
      )
    },
  }}
/>
```
